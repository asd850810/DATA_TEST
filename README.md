# SPC 數據即時分析軟體

這是一款使用 Python 和 PyQt5 開發的即時 SPC（統計製程管制）數據分析工具。它可以監控指定資料夾中的量測數據檔案（支援 .txt, .csv, .log），並自動載入最新的檔案進行分析與視覺化。

## 主要功能

- **即時檔案監控**: 自動偵測並載入指定資料夾中的最新數據檔案。
- **多樣化圖表分析**:
    - **趨勢圖**: 顯示各量測項目的數據走勢，並可自訂抽樣率。
    - **能力圖 (Capability Analysis)**: 包含直方圖、核密度估計（KDE），並計算 Cpk、Cp 等製程能力指標。
    - **分類長條圖**: 對於文字型態的數據（如 "OK", "NG"），以長條圖顯示分佈。
- **互動式操作**:
    - 點擊小圖可切換主顯示區的詳細圖表。
    - 拖放小圖可自訂排序。
    - 在趨勢圖上框選區間可對局部數據進行分析。
- **高度客製化**:
    - 可編輯各量測項目的上下限（USL/LSL），並支援依不同「機種」設定。
    - 可自訂停機、休息時間的判斷標準。
    - 可自訂良率（Yield）計算的關鍵字。
    - 可自訂圖表顏色。
- **摘要與匯出**:
    - 提供全資料夾的檔案摘要功能，快速計算各檔案的稼動率、UPH、良率等指標。
    - 摘要結果可匯出為 CSV 檔案。

## 安裝

1.  確認您已安裝 Python 3.8 或更高版本。
2.  複製此專案到您的本地電腦。
3.  開啟終端機或命令提示字元，進入專案根目錄。
4.  執行以下指令安裝必要的相依套件：

    ```bash
    pip install -r requirements.txt
    ```

## 如何執行

進入專案根目錄後，執行以下指令啟動應用程式：

```bash
python run.py
```

程式啟動時會要求您選擇一個包含數據檔案的資料夾。選擇後，程式將會自動載入最新的檔案並開始分析。

## 專案結構

- `run.py`: 應用程式的進入點。
- `requirements.txt`: 專案的相依套件列表。
- `src/`: 包含所有原始碼。
  - `core/`: 核心邏輯模組（設定、資料處理、統計計算等）。
  - `dialogs/`: 各式設定與互動對話視窗。
  - `widgets/`: 自訂的 PyQt5 元件（圖表、開關等）。
  - `main_window.py`: 主視窗的定義。
- `legacy/`: 包含重構前的舊版程式碼備份。
